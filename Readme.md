##图书馆管理系统 
> 详细设计请查看.doc文件

###系统需求
根据图书馆管理系统的特点，将其分为管理员管理、读者管理、图书管理、图书借还4个部分

###功能需求
通过对一些大型图书馆的实际调查与分析，要求本系统必须具备以下功能：<br />
用户界面友好，功能全面，操作简单。<br />
要求对所有读者进行管理。<br />
提供快速的图书信息，实现综合条件查询，保证数据查询的灵活性。<br />
实现图书借阅、图书续借、图书归还功能。<br />
实现图书借阅、续借和归还过程的全程数据信息跟踪。 <br />
提供借阅到期提醒功能，管理者能及时了解到已经超期的图书借阅信息。<br />
具有易维护性和易操作性。<br />
系统运行稳定、安全可靠。<br />
